# Этап 1: OSINT

## Используемые источники

- **Google (google.com) и техники Google Dorking** — поисковая система общего назначения, применяемая в OSINT для выявления публично доступных материалов, связанных с целевым сервисом: упоминания доменов/поддоменов, техническая документация, следы конфигураций, ссылки на административные интерфейсы, индексируемые директории, а также потенциальные признаки утечек или ошибок публикации. Использование операторов расширенного поиска позволяет точнее находить релевантные результаты; все выводы, полученные из поисковой выдачи, рассматриваются как предположения до подтверждения на последующих этапах.
- **Shodan (shodan.io)** — специализированная поисковая система по устройствам и сетевым сервисам, доступным из сети Интернет. В рамках OSINT применяется для получения первичных сведений о публичной поверхности: обнаружение доступных сервисов, анализ баннеров, сетевых метаданных и косвенных признаков используемых технологий. Результаты Shodan используются как исходные гипотезы и требуют последующего подтверждения активными методами (этап Scanning/Testing).
- **Criminal IP (criminalip.io)** — сервис киберразведки (CTI), предоставляющий информацию о сетевых активах и связанных с ними рисках на основе наблюдаемых признаков (например, открытые порты/сервисы, технологические отпечатки, индикаторы возможных уязвимостей). В работе используется для расширения контекста по цели тестирования и для формирования списка потенциально рискованных компонентов, которые далее валидируются в рамках активного обследования и ручного тестирования.
